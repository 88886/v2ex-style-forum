## Deployment
```shell script
git clone https://github.com/songquanpeng/v2ex.git
cd v2ex
npm install
npm run build  # For Windows user, please run `npm run build2` instead
npm start
```

How to update it?
```shell script
git pull
npm install
npm run migrate
```
